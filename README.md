# 🎹 NEON_DAW

A high-performance, browser-based Digital Audio Workstation (DAW) built with **Next.js**, **TypeScript**, and the **Web Audio API**. NEON_DAW features a sleek, neon-inspired interface with real-time frequency visualization and precise audio control.
